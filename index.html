啟用「真正多人即時共編」設定步驟

這份簡報採用 Firebase Realtime Database（Google 提供，免費額度足夠此用途）作為同步後端。請依下列步驟取得設定資料，貼到下方欄位即可：

  1. 開啟 https://console.firebase.google.com，用 Google 帳號登入，點「新增專案」，輸入專案名稱（例如 115期中簡報），一路下一步建立完成。
  2. 左側選單「建構」→「Realtime Database」→「建立資料庫」，地區選 asia-southeast1 或預設即可，安全性規則先選「測試模式」（方便所有老師讀寫，30天後需手動延長或改規則）。
  3. 左側「專案總覽」旁的齒輪 ⚙ →「專案設定」，往下捲到「您的應用程式」，點 </>（網頁）圖示新增一個應用程式，命名任意即可。
  4. 建立後會顯示一段 firebaseConfig = {...} 的程式碼，把整段（含大括號內容）複製，貼到下方欄位，按「儲存設定」。
  5. 儲存後重新整理頁面即可生效。把這個 HTML 檔上傳到 GitHub Pages 或 Netlify 取得網址後，分享給其他老師，大家打開同一網址編輯即會即時同步。
